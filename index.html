Investigation Into The Structure of Methylammonium Lead Iodide

3rd Year Research Project

Electron Monopole interactions Along a Two Dimensional Plane

The trajectories of electrons confined to a two dimensional electron gas (2DEG) and situated above a magnetic monopole were simulated. For this purpose a computer model was constructed using the programming language Fortran 95.

Neural Network Classification Project

A neural network was built to classify between two groups of data, a number of data set types were used to test the network with different network topologies. The backpropagation algorithm was used to train the neural network over a number of epochs.
